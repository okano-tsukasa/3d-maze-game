================================================================================
1. プロジェクト概要
================================================================================

1.1 プロジェクト名
    3D Maze Game（3D迷路ゲーム）

1.2 目的
    C言語とraylibライブラリを使用して、GitHubで公開可能な
    教育的かつ健全な3D迷路ゲームを開発する。

1.3 対象プラットフォーム
    - Windows 10/11
    - MSYS2環境でのビルド

1.4 使用技術
    - プログラミング言語: C言語
    - グラフィックスライブラリ: raylib
    - コンパイラ: GCC (MSYS2)

================================================================================
2. 機能要件
================================================================================

2.1 必須機能

2.1.1 3D迷路の表示
    - 一人称視点（FPS視点）での3D迷路の描画
    - 壁と床の視覚的な区別
    - スムーズな3Dレンダリング

2.1.2 プレイヤー操作
    - キーボードによる移動操作
      * W キー: 前進
      * S キー: 後退
      * A キー: 左移動
      * D キー: 右移動
    - マウスによる視点操作
      * マウス移動で視点の回転

2.1.3 ゲームロジック
    - スタート地点の設定
    - ゴール地点の設定
    - ゴール到達時のクリア判定
    - 壁との衝突判定

2.1.4 UI表示
    - 操作説明の表示
    - ゴール到達時のメッセージ表示
    - FPS（フレームレート）の表示

2.2 追加機能（今回は実装しない）
    - タイマー機能
    - スコアシステム
    - 複数レベル
    - セーブ/ロード機能

================================================================================
3. 非機能要件
================================================================================

3.1 パフォーマンス
    - 60FPS以上での動作
    - スムーズなカメラ移動

3.2 操作性
    - 直感的なキーボード・マウス操作
    - レスポンシブな操作感

3.3 保守性
    - 読みやすいコード構造
    - 適切なコメント
    - モジュール化された設計

3.4 ライセンス
    - オープンソース（MIT License推奨）
    - 悪用のリスクがないコード

================================================================================
4. システム要件
================================================================================

4.1 開発環境
    - OS: Windows 10/11
    - コンパイラ: GCC 15.2.0以上
    - ライブラリ: raylib 5.0以上
    - ビルドツール: MSYS2

4.2 実行環境
    - OS: Windows 10/11
    - メモリ: 512MB以上
    - グラフィックス: OpenGL 3.3対応GPU

================================================================================
5. 画面仕様
================================================================================

5.1 ゲーム画面
    - ウィンドウサイズ: 800x600ピクセル
    - タイトル: "3D Maze Game"
    - 3D迷路の描画領域
    - 操作説明テキストの表示領域

5.2 表示内容
    - 迷路の壁（グレー）
    - 迷路の床（ダークグレー）
    - ゴール地点（緑色の立方体）
    - 操作説明テキスト
    - FPS表示

================================================================================
6. 迷路仕様
================================================================================

6.1 迷路サイズ
    - 10x10のグリッド
    - 各セルのサイズ: 2.0ユニット

6.2 迷路構造
    - 壁（1）と通路（0）で構成
    - スタート地点: (1, 1)
    - ゴール地点: (8, 8)
    - クリア可能な経路が存在すること

7.1 技術的制約

7.2 セキュリティ
    - 悪意のあるコードを含まない

    - 個人情報を収集しない
